## 遇到的问题和解决

1. 前后端字段命名不一致，导致页面无法正确展示数据。解决方式是统一 DTO/VO 字段，并在接口文档中明确请求和响应结构。
2. 部分接口返回格式不统一，前端处理较麻烦。解决方式是使用统一的 `Result<T>` 返回结构，统一包含 code、message、data。
